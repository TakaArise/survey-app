# レビュー設計アンケートメーカー

「こんなレビューがほしい」を選ぶだけで、意図したフィードバックを引き出すアンケート設問を自動生成し、集めた回答を「読まれる販促素材」に変換するアプリ。

## 使い方

`index.html` をブラウザで開くだけ（サーバー不要・インストール不要）。

### モード1：レビュー収集（利用後アンケート）
1. 商品・サービス名と概要を入力
2. ほしいレビューの種類をチェック（導入前の悩み／時間の変化／お客様の声／価格の妥当性など10種類）
3. 「アンケートを生成する」→ 設問リストが出る
4. 不要な設問は ✕ で外す → 「送付用テキスト」をコピーしてGoogleフォームやLINEに貼り付け

### モード2：不満募集キャンペーン（見込み客リスト獲得）
競合サービスや「今のやり方」への不満を募集するキャンペーン用アンケートを生成。
「不満の言語化 → 深刻度 → 払える金額 → 連絡先」の流れでリスト獲得に最適化。
集まった不満は投票でランキング化して発表する前提の文面（キャンペーン設計）。

### Googleフォーム連携
- **Gフォーム変換**：生成したアンケートから、Googleフォームを自動作成するスクリプト（Google Apps Script）を出力。script.google.com に貼って実行するだけでフォームが完成
- 回答はフォームにリンクした回答スプレッドシートに自動で蓄積（これが回答データベース）
- 回答シートを全選択コピー→素材変換欄に貼り付けると、回答件数を自動検出

### 回答→販促素材に変換（AIモード）
回収した回答を貼り付けると、以下を自動生成：
- **反響トップ5**：成功事例をランキング形式に（「人気の口コミ」として読まれる見せ方）
- **購入直後の声トップ5**：申込直後の生の一言をランキング化（鮮度＝リアリティ）
- **紹介キャッチフレーズ集**：「誰に・どんな一言で」の紹介台本＋ビフォーアフター型コピー
- **改善点ランキング（公開用）**：不満を隠さず公開し対応方針を添えて信頼をつくる

捏造禁止・原文の言葉優先・名前はプレースホルダ（掲載許諾確認後に記入する運用）。

## アプリならではの機能

- **活用先タグ**：各設問に「この回答が何に使えるか」（LP実績数値／お客様の声／ランキング素材／リスト獲得／価格設定／改善リスト／信頼性担保）を表示
- **あいまい設問ガード**：時間・数値系の設問に「初期設定の時間は含めない」「未経験の場合の答え方」などの注記を自動付与し、回答の矛盾を防ぐ
- **回答負担メーター**：設問数から推定回答時間を算出し、離脱リスクを警告
- **掲載許諾セクション自動付与**：SNS・LP掲載に必要な許諾（写真掲載可否含む）を忘れず入れる
- **AIモード（任意）**：設問文・例文を商品に合わせて最適化（claude-sonnet-4-6 / max_tokens 8000）、回答→素材変換（claude-sonnet-4-6 / max_tokens 6000）

## セキュリティ

- APIキーはブラウザのlocalStorageにのみ保存。外部送信はAnthropic APIへのリクエストのみ（CSPで通信先をapi.anthropic.comに制限）
- キー入力欄は伏せ字表示。共用PC向けに「キー削除」ボタンあり
- ユーザー入力・AI出力はすべてエスケープして描画（XSS対策）
- サーバーなしの静的ページのため、運営側にキーや回答データは一切渡らない
