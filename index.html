📋 レビュー設計アンケートメーカー

「こんなレビューがほしい」を選ぶだけで、意図したフィードバックを引き出すアンケート設問を自動生成します

1. 商品・サービスの情報

2. どんなレビュー・フィードバックがほしい？ （複数選択）

不満募集キャンペーンでは、「不満の言語化 → 深刻度 → 払える金額 → 連絡先（リスト獲得）」の流れに最適化した設問セットを自動で組みます。集まった不満は投票でランキング化して発表する前提の文面になります。

🤖 AIモード（任意）：Claude APIキーを入れると、生成した設問の文面・例文をあなたの商品に合わせて最適化します。キーはこのブラウザ内（localStorage）にのみ保存され、送信先はAnthropic APIだけです。共用のパソコンでは、使い終わったら「キー削除」を押してください。

3. 集めた回答 → 販促素材に変換 🤖

回収した回答を貼り付けると、AIが「読まれる見せ方」（ランキング形式など）に整形します。上のAIモードのAPIキーを使います。
Googleフォームで集めた場合：フォームの「回答」タブ→スプレッドシートを開き、全選択（Ctrl+A / Cmd+A）してコピー→そのままここに貼り付けでOKです。

設問数 0問
推定回答時間 0分
左側で情報を入力して「アンケートを生成する」を押してください。

生成された設問には「この回答が何に使えるか」のタグが付きます。

📖 使い方ガイド

      
    

    

      

        このツールは 「アンケートを作る → お客様に送る → 集めた回答を販促素材に変える」 までを一気通貫でサポートします。基本は無料で使え、AI機能（設問の最適化・素材変換）だけ Claude の API キーが必要です。
      


      

        入力した内容は、このブラウザの中だけで扱われます（外部のサーバーには保存されません）。そのぶん、ページを閉じたり再読み込みしたりすると入力内容は消えます。作ったアンケートや素材は、閉じる前に右上の 「📋 コピー」または「💾 保存」 をしておいてください。
      


      
基本の流れ（3ステップ）


      

        
1

        

          
アンケートを作る

          
左側で商品・サービス名を入れ、「どんなレビューがほしいか」をチェックして 「アンケートを生成する」 を押します。設問が右側に出ます。

          
・各設問には「この回答が何に使えるか」のタグ（LP実績数値・お客様の声・価格設定など）が付きます。
・不要な設問は設問カード右上の ✕ で外せます。
・上部の「回答負担メーター」で、設問数と答えるのにかかる時間の目安が分かります。

        

      


      

        
2

        

          
お客様に送る

          
右上のタブで送り方を選びます。

          

            ・送付用テキスト：そのままコピーして、LINE やメールに貼り付け。

            ・Gフォーム変換：Google フォームを自動で作るスクリプトが出ます（下の「Googleフォームで集める」を参照）。

            右上の 📋 コピー / 💾 保存 で、表示中の内容をコピー・ダウンロードできます。
          

        

      


      

        
3

        

          
集めた回答を販促素材に変える

          
お客様から返ってきた回答を、左下の 「3. 集めた回答 → 販促素材に変換」 に貼り付けて生成すると、AI が「読まれる見せ方」に整形します。

          

            作れる素材：反響トップ5（成功事例をランキング化）／購入直後の声トップ5（申込直後の生の一言）／紹介キャッチフレーズ集（誰に・どんな一言で）／改善点ランキング（不満を公開して信頼をつくる）。

            ※この機能は AI モード（API キー）が必要です。
          

        

      


      
Googleフォームで集める

      

        Gフォーム変換タブのスクリプトを使うと、設問を1問ずつコピペせずにフォームが作れます。


        ① 右上「Gフォーム変換」タブを開き、📋 コピー

        ② script.google.com を開き「新しいプロジェクト」→ 最初のコードを消して貼り付け

        ③ 「実行」ボタンを押す（初回は Google の確認 →「許可」）

        ④ 画面下のログに出る 回答用URL をお客様に送るだけ


        回答は自動でスプレッドシートに溜まります（＝これが回答データベース）。集まったら、そのシートを全選択コピー（Ctrl+A / Cmd+A）して、ステップ3の貼り付け欄に入れると、回答件数が自動で表示されます。
      


      
2つのモード

      

        
A

        

          
レビュー収集（利用後アンケート）

          
使ってくれたお客様から、LP やお客様の声に使えるフィードバックを集めます。通常はこちら。

        

      

      

        
B

        

          
不満募集キャンペーン

          
「今のやり方・競合サービスへの不満」を募集して、まだお客様でない人のリストを集めます。「アンケートの目的」で切り替え。集客の入口として使えます。

        

      


      
AIモード（任意）について

      

        Claude の API キーを入れると「設問文の最適化」と「素材変換」が使えます。キーは このブラウザ内にのみ保存され、送信先は Anthropic の API だけです。共用のパソコンでは、使い終わったら 「キー削除」 ボタンを押してください。API キーは console.anthropic.com で取得できます（利用分はキーの持ち主の負担になります）。
      


      

        困ったとき

        AIボタンを押してエラーが出たときは、まず API キーが正しく入力されているか確認してください。それでも直らない場合は、インターネットにつながっているか確認し、少し時間をおいてもう一度お試しください。
